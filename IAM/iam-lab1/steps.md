# IAM Lab 1 — Steps

## Step 1: Create IAM User
- Created IAM user: `dev-user`
- Enabled AWS Management Console access

## Step 2: Create IAM Group
- Created group: `developers`

## Step 3: Attach Policy to Group
- Attached policy: `AmazonS3ReadOnlyAccess`

## Step 4: Add User to Group
- Added `dev-user` to `developers` group

## Step 5: Test Access
- Logged in as IAM user
- Accessed S3 service successfully (read-only access confirmed)
