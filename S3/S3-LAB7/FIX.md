# Fix — Securing the Bucket

## 🔧 Step 1: Enable Block Public Access
- Turn ON all block public settings

## 🔧 Step 2: Remove Public Policy
- Delete or restrict bucket policy

## 🔧 Step 3: Remove Public Object Access
- Set object back to private

## 🔧 Step 4: Apply IAM Control
- Allow only specific IAM user/role access

## 🧠 Principle Applied
Least privilege — only authorized users can access data
