# S3 Lab — Public vs Private Access

## 🎯 Objective
To demonstrate how S3 objects can be private by default and how misconfiguration can expose them to the public.

## 🧪 Scenario
A file stored in S3 is initially private and inaccessible via URL, but becomes publicly accessible after changing permissions.

## ☁️ AWS Service Used
- S3

## 🔍 Focus
- Object-level permissions
- Public access exposure
- Access via URL

## 🧠 Key Idea
S3 objects are private by default, but can be exposed if public access is enabled.

## ⚠️ Risk
Public access allows anyone on the internet to retrieve sensitive data if misconfigured.
