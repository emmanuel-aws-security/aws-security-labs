# S3 Lab — Basic Operations

## 🎯 Objective
To demonstrate basic interaction with S3 by storing, retrieving, and deleting objects.

## 🧪 Scenario
A user uploads a file to S3, retrieves it, and removes it when no longer needed.

## ☁️ AWS Service Used
- S3 (Simple Storage Service)

## 🔍 Focus
- Bucket creation
- Object upload/download
- Basic storage operations

## 🧠 Key Idea
S3 is used to store and retrieve files reliably from the cloud.
