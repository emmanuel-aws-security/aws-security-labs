# S3 Lab — Exposure Attack Scenario

## 🎯 Objective
To simulate how misconfigured S3 permissions can lead to public data exposure.

## 🧪 Scenario
A sensitive file is stored in an S3 bucket with public read access enabled, making it accessible to anyone with the URL.

## ☁️ AWS Service Used
- S3

## 🔍 Focus
- Public access misconfiguration
- Data exposure risk
- Least privilege principle

## 🧠 Key Idea
If S3 is made public, data is exposed instantly without authentication.

## ⚠️ Risk
This is one of the most common real-world cloud security breaches.
