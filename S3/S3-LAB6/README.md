# S3 Lab — Logging & Visibility

## 🎯 Objective
To understand how S3 access can be tracked using logging features.

## 🧪 Scenario
A file in S3 is accessed, and we check how AWS records or tracks that access.

## ☁️ AWS Service Used
- S3

## 🔍 Focus
- Basic access logging awareness
- Visibility of object access
- Audit concept

## 🧠 Key Idea
S3 logging helps answer the question: "Who accessed this data?"
