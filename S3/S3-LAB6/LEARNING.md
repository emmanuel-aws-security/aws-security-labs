# S3 Lab — Logging Learnings

## 🔴 Core Concept
S3 logging records access activity for audit and monitoring purposes.

## 🧠 Key Insight
Logging helps answer:
- who accessed the file
- when it was accessed
- what action was performed

## 🔍 What I Understood
- S3 itself does not “block” or “allow” using logs
- Logs are for visibility only

## ⚠️ Important Note
Logging is NOT security enforcement—it is observation.

## 🧾 Final Conclusion
Visibility is critical for detecting unauthorized access after it happens.
