# S3 Lab — Bucket Policy

## 🎯 Objective
To understand how bucket policies control access to S3 resources using JSON.

## 🧪 Scenario
A bucket is made publicly accessible through a policy and later secured by removing that access.

## ☁️ AWS Service Used
- S3

## 🔍 Focus
- Bucket policy structure
- Access control using JSON
- Public exposure via policy

## 🧠 Key Idea
Bucket policies define who can access what in S3. Misconfiguration can expose entire buckets.

## ⚠️ Risk
A single incorrect policy can make all objects publicly accessible.
