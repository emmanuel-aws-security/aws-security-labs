# VPC Lab — Flow Logs (Traffic Visibility)

## 🎯 Objective
To enable VPC Flow Logs and understand how AWS records network traffic for monitoring and security analysis.

## 🧪 Scenario
Traffic is generated between instances and observed as ACCEPT or REJECT events in Flow Logs.

## ☁️ AWS Services Used
- VPC
- Flow Logs
- CloudWatch Logs (or S3)

## 🔍 Focus
- Network traffic visibility
- ACCEPT vs REJECT behavior
- Security monitoring

## 🧠 Key Idea
If you cannot see traffic, you cannot secure it. Flow Logs provide visibility into network-level activity.
