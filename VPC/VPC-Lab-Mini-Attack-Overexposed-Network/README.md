# VPC Mini Attack Scenario — Overexposed Network

## 🎯 Objective
To simulate a misconfigured VPC where security boundaries are weak, and demonstrate how exposure leads to attack risk.

## 🧪 Scenario
A public subnet is misconfigured with overly open access rules and no proper segmentation.

## ☁️ AWS Services Used
- VPC
- Subnets
- Security Groups
- EC2

## 🔍 Focus
- Exposure risks
- SSH misconfiguration
- Lack of network segmentation

## 🧠 Key Idea
If SSH is open to the world and everything is in a public subnet, the entire environment is exposed by default.
