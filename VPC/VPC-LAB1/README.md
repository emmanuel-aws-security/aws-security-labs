# VPC Lab 1 — Custom VPC Build (Foundation)

## 🎯 Objective
To build a custom VPC from scratch and demonstrate the difference between public and private subnets using routing and internet access.

## 🧪 Scenario
A network is designed with two subnets:
- Public subnet → should have internet access
- Private subnet → should NOT have internet access

## ☁️ AWS Services Used
- VPC
- Subnets
- Internet Gateway
- Route Tables
- EC2 (for testing)

## 🔍 Focus
- Manual VPC creation
- Subnet isolation
- Internet routing behavior

## 🧠 Key Idea
Internet access in AWS is controlled by routing and gateways—not by the subnet name.
