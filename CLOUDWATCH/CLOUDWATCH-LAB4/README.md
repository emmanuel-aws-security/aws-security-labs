# CloudWatch + CloudTrail Integration (Awareness Lab)

## 🎯 Objective
To understand how CloudTrail and CloudWatch work together for detection and monitoring in AWS.

## 🧪 Scenario
An AWS account activity occurs (e.g., root login or sensitive action), and we explore how:
- CloudTrail records the event
- CloudWatch can trigger alerts based on patterns

## ☁️ AWS Services Used
- CloudTrail
- CloudWatch
- IAM

## 🔍 Focus
- Audit vs Monitoring
- Detection awareness
- Basic security signal understanding

## 🧠 Key Idea
CloudTrail tells you WHAT happened. CloudWatch helps you react to patterns or abnormal behavior.
