# CloudWatch Lab — Basic Log Monitoring

## 🎯 Objective
To view and understand basic log monitoring using CloudWatch Logs.

## 🧪 Scenario
An EC2 instance generates system/application logs, and we inspect them for visibility and troubleshooting.

## ☁️ AWS Services Used
- EC2
- CloudWatch Logs

## 🔍 Focus
- Viewing logs
- Understanding centralized logging
- Basic troubleshooting awareness

## 🧠 Key Idea
Logs help you understand what is happening inside a system when things go wrong.
