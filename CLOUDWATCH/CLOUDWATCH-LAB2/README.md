# CloudWatch Lab — Alarms & Detection

## 🎯 Objective
To create CloudWatch alarms that detect unhealthy EC2 behavior and performance problems.

## 🧪 Scenario
An EC2 instance experiences abnormal CPU activity or failed health checks, triggering monitoring alerts.

## ☁️ AWS Services Used
- EC2
- CloudWatch

## 🔍 Focus
- CPU alarms
- Instance status alarms
- Basic detection visibility

## 🧠 Key Idea
CloudWatch alarms help detect unhealthy or suspicious system behavior automatically.

## ⚠️ Importance
Without alarms:
- problems may go unnoticed
- outages may last longer
- abnormal behavior may not be detected quickly
