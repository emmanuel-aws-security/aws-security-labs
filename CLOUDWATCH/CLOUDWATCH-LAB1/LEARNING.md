# CloudWatch Lab — Learnings

## 🔴 Core Concept
CloudWatch monitors AWS resource health and performance metrics.

## 🧠 What I Learned

### CPUUtilization
Shows how heavily the EC2 instance is being used.

### Status Checks
AWS performs health checks for:
- hardware
- networking
- instance reachability

### CloudWatch Alarms
Alarms help detect:
- unhealthy systems
- abnormal behavior
- performance problems

## 🔍 Key Insight
Monitoring helps identify problems before systems completely fail.

## ⚠️ Security Importance
Unusual CPU spikes may indicate:
- abuse
- runaway processes
- resource exhaustion
- suspicious activity

## 🧾 Final Conclusion
CloudWatch provides operational visibility into EC2 health and performance.
