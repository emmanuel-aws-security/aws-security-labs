# CloudWatch Lab — EC2 Monitoring

## 🎯 Objective
To monitor EC2 instance health and basic performance metrics using CloudWatch.

## 🧪 Scenario
An EC2 instance is monitored to identify unhealthy behavior and performance issues.

## ☁️ AWS Services Used
- EC2
- CloudWatch

## 🔍 Focus
- CPU monitoring
- Instance status checks
- Basic alert visibility

## 🧠 Key Idea
CloudWatch helps monitor system health and detect operational problems inside AWS infrastructure.

## ⚠️ Importance
Without monitoring:
- unhealthy systems may go unnoticed
- failures may not be detected quickly
