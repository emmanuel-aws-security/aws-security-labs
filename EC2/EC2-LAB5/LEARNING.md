# EC2 Lab 7 — Learnings

## 🔍 Core Concept
Logs provide visibility into what is happening on a server.

## 💥 Real-World Use
In a security incident, logs help answer:
- Who accessed the server?
- When did they access it?
- What did they do?

## 📜 Authentication Logs
- Show login attempts
- Show successful vs failed access
- Can reveal suspicious activity

## 🧠 Command History Insight
- Shows actions performed by users
- Useful for tracing changes made on the system

## ⚠️ Limitations
- History can be cleared
- Logs can be rotated or deleted
- Not all actions are always recorded

## 🔧 Security Insight
Logs should be monitored and stored securely to prevent tampering.

## 🧾 Final Conclusion
Without logs, detecting unauthorized activity becomes difficult or impossible.
