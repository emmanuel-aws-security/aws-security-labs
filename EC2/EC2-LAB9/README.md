# EC2 Debug Lab — Case 4: Permission Errors

## 🎯 Objective
To identify and fix access issues caused by incorrect file ownership and permissions.

## 🧪 Scenario
The web server is running, but content is not being served due to permission issues.

## ☁️ Environment
- EC2 (Ubuntu)
- nginx web server

## 🔍 Focus
- File ownership
- File permissions
- Access control at OS level

## 🧠 Key Idea
Even if the server and network are correct, improper permissions can silently break access.
