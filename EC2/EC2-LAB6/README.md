# EC2 Debug Lab — Case 1: Web Server Down

## 🎯 Objective
To simulate a web server failure and demonstrate how to detect and recover from the issue.

## 🧪 Scenario
A web server becomes inaccessible because the service has stopped running.

## ☁️ Environment
- EC2 (Ubuntu)
- nginx web server

## 🔍 Focus
- Service status checking
- Service recovery
- Basic failure detection

## 🧠 Key Idea
If a web service is not running, the server may appear “down” even though the instance itself is healthy.
