# EC2 Real-World Scenario — Compromised Server Simulation

## 🎯 Objective
To simulate a real EC2 security breach scenario and demonstrate detection, investigation, and remediation.

## 🧪 Scenario
A misconfigured EC2 instance is exposed to the internet with weak security controls and sensitive data.

## ☁️ Environment
- EC2 (Ubuntu)
- SSH access
- nginx web server
- AWS CLI (optional)

## 🔍 Focus
- Attack surface exposure
- Credential leakage risk
- Incident investigation
- Security remediation

## 🧠 Key Idea
Security is not about deployment—it is about controlling exposure, detecting compromise, and fixing root causes.
