# EC2 Lab 1 — Instance Setup & Access Verification

## 🎯 Objective
To launch an EC2 instance, connect to it using SSH, and deploy a basic web service to verify connectivity.

## 🧪 Scenario
A user deploys a server in AWS and confirms it is reachable both internally and externally.

## ☁️ AWS Services Used
- EC2

## 🔍 Focus
- Instance creation
- SSH access using key pairs
- Basic service deployment (nginx)
- Connectivity testing

## 🧠 Key Idea
Launching an EC2 instance is not enough. You must verify that it is accessible and functioning as expected.
