# CloudTrail Lab — Track Basic Actions

## 🎯 Objective
To track AWS actions using CloudTrail and identify who performed them.

## 🧪 Scenario
An object is deleted from S3, and CloudTrail is used to investigate the action.

## ☁️ AWS Services Used
- CloudTrail
- S3
- IAM

## 🔍 Focus
- Event visibility
- User activity tracking
- Basic investigation

## 🧠 Key Idea
CloudTrail records API activity, making it possible to trace actions performed in AWS.

## ⚠️ Importance
Cloud security depends heavily on visibility and traceability.
