# CloudTrail Lab — S3 Data Events

## 🎯 Objective
To enable S3 Data Events in CloudTrail and track object-level access activity.

## 🧪 Scenario
A file inside an S3 bucket is accessed, and CloudTrail is used to identify who accessed it.

## ☁️ AWS Services Used
- CloudTrail
- S3

## 🔍 Focus
- Object-level visibility
- S3 Data Events
- Access investigation

## 🧠 Key Idea
CloudTrail management events alone do not provide full visibility into S3 object access activity.

## ⚠️ Importance
Without S3 Data Events enabled, object access actions like GetObject may not be fully visible during investigations.
