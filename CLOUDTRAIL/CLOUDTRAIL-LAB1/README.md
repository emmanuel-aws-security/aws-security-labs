# CloudTrail Lab — Enable Logging

## 🎯 Objective
To enable AWS CloudTrail and confirm that AWS account activity is being recorded and stored.

## 🧪 Scenario
CloudTrail is configured to log AWS activity into an S3 bucket for visibility and auditing.

## ☁️ AWS Services Used
- CloudTrail
- S3

## 🔍 Focus
- Account activity logging
- Centralized log storage
- Visibility into AWS actions

## 🧠 Key Idea
CloudTrail records AWS API activity, helping security teams investigate actions performed inside the account.

## ⚠️ Importance
Without logging, security incidents cannot be investigated properly.
