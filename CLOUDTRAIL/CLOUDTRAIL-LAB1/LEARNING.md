# CloudTrail Lab — Learnings

## 🔴 Core Concept
CloudTrail records AWS account activity and API actions.

## 🧠 What I Learned
- CloudTrail stores logs in S3
- Events contain:
  - user identity
  - action performed
  - timestamp
  - source IP

## 🔍 Visibility Gained
I can trace actions performed inside the AWS account.

## ⚠️ Security Importance
Without CloudTrail:
- actions cannot be audited
- investigations become difficult

## 🧾 Final Conclusion
CloudTrail is foundational for cloud visibility and incident investigation.
