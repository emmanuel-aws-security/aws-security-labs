# IAM Lab 7 — Account Security Basics (MFA & Password Policy)

## 🎯 Objective
To secure the AWS root account using MFA and enforce strong password policies for IAM users.

## 🧪 Scenario
The root account is secured with Multi-Factor Authentication (MFA), and an account password policy is configured to improve security.

## ☁️ AWS Services Used
- IAM
- AWS Account Settings

## 🔍 Focus
- Root account protection
- Multi-Factor Authentication (MFA)
- Password policy enforcement

## 🧠 Key Idea
Password-only authentication is not enough to secure AWS accounts.
